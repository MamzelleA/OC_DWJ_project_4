Ce code est réalisé dans le cadre du projet 4 de la formation Développeur Web Junior d'Openclassrooms.

Ce projet a pour but de créer un blog pour un écrivain en utilisant :

Langages et frameworks utilisés : HTML5, CSS3 et Bootstrap, Javascript et jQuery, PHP7, Bases de données et SQL 

Le projet doit comporter une partie pour les utilisateurs et un espace d'administration protégée par mot de passe. L'administrateur doit pouvoir modérer les commentaires, rédiger et modififier ses articles selon une interface WYSIWYG.

Le projet doit être développé selon une architecture MVC et en POO.

------------------------------------------------------------------------------------------------------------------EN

This code is produced as part of Project 4 of the Openclassrooms Junior Web Developer Training.

This project aims to create a blog for a writer using:

Using langages and frameworks : HTML5, CSS3 and Bootstrap, Javascript and jQuery, PHP7, Databases and SQL

The project must include a portion for users and a password-protected administration area. Users should be able to leave comments and flag those that seem inappropriate to them. The administrator must be able to moderate the comments, write and modify his articles according to a WYSIWYG interface.

The project must be developed according to an MVC and OOP architecture.
